Fainting Goat Web Server

This is a "minimum viable web server" -- less than 100 lines of framework-free JavaScript meant to be run in a NodeJS environment. As the name implies, it is designed to fall over upon any error. It is a "canary in the coalmine" to verify that your standards-based, browser-native web app is free of errors that might otherwise be hidden or surpressed by production-caliber web servers.
